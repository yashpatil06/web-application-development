 Step 2: Install Apache Web Server
Once you're in the EC2 terminal, run:


    sudo yum update -y
    sudo yum install httpd -y
    sudo systemctl start httpd
    sudo systemctl enable httpd


Make /var/www/html Writable:

    sudo chmod 777 /var/www/html
